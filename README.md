## What is spendora?

Spendora is a JavaScript application built with React and Vite. 
It utilizes recharts for data visualization and gh-pages for deployment.
The application is licensed under MIT, allowing free use and modification.

Spendora's codebase includes JavaScript, CSS, and HTML files. 
The project structure is organized into src, public, and root directories.

## Features

| Feature | Description |
|---|---|
| **react** | Renders UI components |
| **recharts** | Visualizes data |
| **vite** | Builds and serves application |
| **gh-pages** | Deploys application |
| **css** | Styles UI components |
| **html** | Defines UI structure |
| **javascript** | Handles application logic |
| **eslint** | Enforces code standards |
