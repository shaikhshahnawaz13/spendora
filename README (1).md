## What is spendora?
Spendora is a JavaScript application built using React and Vite. It utilizes recharts for data visualization. 
The application is deployed using gh-pages. 
The code is formatted using eslint.

## Features
| Feature | Description |
|---|---|
| **Build** | Runs vite build |
| **Dev** | Starts development server |
| **Deploy** | Deploys application to gh-pages |
| **Preview** | Previews built application |
| **Predeploy** | Builds application before deployment |
| **React** | Utilizes React framework |
| **Recharts** | Uses recharts for data visualization |

## Tech Stack
| Layer | Technology |
|---|---|
| Frontend | React, recharts |
| Build Tool | Vite |
| Deployment | gh-pages |
| Styling | CSS |
| Scripting | JavaScript |
